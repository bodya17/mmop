# Розподіл балів
    теорія - 30б (20б в білеті, 10 усно)
    практика - 40б

# Практика
    5 завдань тестових

    Типи практичних завдань:
    * Знайти МНК оцінку
    * Перевірити лінійну гіпотезу (треба буде знайти R^2_1, R_0^2)

    * Лінійні репліки (генеруюче співвідношення, визначальне співвідношення)
    * Блок схеми
    * Критерії оптимальності (буде дано 2 плани, порівняти ці плани за критерієм (A, E, G...) )

# Теорія (2 питання)
    * МНК оцінки (знаходження МНК оцінки за означенням)
    * Довести, що МНК оцінка (незміщена, найкраща)
    * Узагальнені МНК оцінки
    * Дисперсії узагальненої оцінки
    * Модель неповного рангу
    * Перевірка гіпотез
    * Репліки
        * визначальне співвідношення
        * генеруюче співвідношення
    * Критерії оптимальності
        * шо таке плани
        * інформаційна матриця
        * дисперійна матриця
